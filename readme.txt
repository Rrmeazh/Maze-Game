Maze Game
========
欢迎游玩本迷宫游戏！以下是本游戏的操作说明和注意事项。

1. 游戏简介

当你开始游戏时，你需要输入迷宫的长度和宽度。然后，本游戏会生成一个随机迷宫。
你可以选择两种迷宫生成方式：
  - 按 'D' 键生成路径更曲折的迷宫。
  - 按 'G' 键生成分岔口更多的迷宫。

迷宫布局如下：
  - '█' 表示墙壁。
  - ' ' （空格）表示可行走的路径。
  - 'S' 表示起点。
  - 'E' 表示终点。
  - '▲' 表示玩家当前的位置。
  - '×' 表示在启用自动寻路功能时的下一步指示标记。

之后，你需要通过键盘输入从起点走到终点：
  - w键 - 向上移动
  - s键 - 向下移动
  - a键 - 向左移动
  - d键 - 向右移动

如果不知道如何通关，可以使用自动寻路功能：
  - h键 - 启用自动寻路功能。
  - n键 - 关闭自动寻路功能。
自动寻路功能启用时，指示标记（'×'）会显示下一步应往何处。

到达终点后，你可以选择重新游玩。
按y键重新开始游戏，按其他任意键退出。

2. 注意事项

输入的长度和宽度应为小于等于50的正整数。
尽量避免移动到墙壁或边界上。
当试图移动到墙壁或边界上，游戏给出提示时，请等待提示消失后再继续操作。
